# APS PERFORMANCE EM C

### For any user:
To compile the code on terminal use, with the Makefile:
```
  make
```
Then execute the file with:
```
  ./mainapp
```

### For Windows users:
To compile the code on terminal use:
```
  gcc -c main.c
```
Then create and execute the .exe file with:
```
  gcc -o main main.o
  start main.exe
```

### For Linux users:
To compile the code on terminal use:
```
  gcc -o main main.c
```
Then execute the file with:
```
  ./mainapp
```
