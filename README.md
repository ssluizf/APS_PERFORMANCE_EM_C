# APS PERFORMANCE EM C

### For Windows users:
To compile the code on terminal use:
```
  gcc -c main.c && gcc -c algorithms.c && gcc -c randomData.c
  gcc -c algorithms.c
  gcc -c randomData.c
```
Then create and execute the .exe file with:
```
  gcc -o main main.o algorithms.o randomData.o
  start main.exe
```
